## 맴버 구성 및 역할 분담
팀장 여수한 : 차트(인기, 최신, 장르별, 앨범)

팀원 김훈규 : 이용권 관리 페이지, 재생목록 페이지

팀원 방수현 : 전체적인 프론트 디자인, 로그인 페이지, 회원가입 페이지

팀원 서지환 : 관리자 페이지, 고객센터 페이지, 회원 관리 페이지, 이용권 구매 기능

팀원 임휘재 : 앨범 페이지, 가수 페이지, 좋아요 기능
